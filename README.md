# Mon Projet GitHub

Ce projet est une expérimentation par des débutants en programmation.  
L’objectif principal était d’apprendre à organiser un projet et à utiliser GitHub.  

Quelques points importants :  
- Certaines parties du code ou du contenu ont été générées avec l’aide d’une intelligence artificielle.  
- Toutes les idées et la structure du projet viennent de nous.  
- Ce dépôt est public pour partager notre apprentissage et recevoir des retours constructifs.  

Merci pour votre compréhension et vos conseils !
